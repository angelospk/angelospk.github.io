# TUI Digital Tools

**Hotel Map & Check-in Automation**

## Overview

A suite of digital tools developed to improve guest experience and operational efficiency 
at TUI Magic Life Plimmiri.

### The Problem
Guests struggled with navigation due to outdated PDF maps, and reception faced long queues 
due to manual data entry requirements.

### The Solution
Developed two solutions: An interactive web map for guests and a Python automation script 
for reception staff.


## Key Features

### Interactive Resort Map
- **Mobile-First PWA**: Responsive map with pin-drop navigation.
- **Google Maps Integration**: One-tap deep-linking for directions.
- **Offline Capability**: Works reliably on spotty hotel Wi-Fi.

### Check-in Automation
- **3x Faster Processing**: Significantly reduced guest wait times.
- **Error Elimination**: Replaced manual typing with automated backend data population.
