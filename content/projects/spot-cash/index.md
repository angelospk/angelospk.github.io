# SpotifyPay

**Spotify Family Cost Splitter**

## Overview

Εφαρμογή υπολογισμού χρεώσεων για Spotify Family subscription.
Υπολογίζει αυτόματα τι χρωστάει κάθε μέλος λαμβάνοντας υπόψη 
αυξήσεις τιμών και στέλνει email ενημερώσεις.
Built with React, Vite, and Shadcn UI.

## Key Features

- Complex State Management
- Interactive Visualizations
- Modern UI/UX
- Serverless Backend
